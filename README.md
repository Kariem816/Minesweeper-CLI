# Minesweeper CLI

A simple game with NodeJS and TypeScript.

## How To Play

1. Clone the repository.
2. Install packages
   
   ```console
   $ npm i
   ```
   
3. Build the app (optional but recommended)

   ```console
   $ npm run build
   ```

4. Run the app using
   
   ```console
   $ npm start
   ```
   
   or if you skipped building

   ```console
   $ npm run dev
   ```
   
5. Enjoy!
